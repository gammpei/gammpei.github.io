How to write a Game Boy emulator (Table of contents)

Posted on 2017-01-07

This is the first post of a blog series about writing a Game Boy emulator.

Writing an emulator may seem like an impossible task, but I can assure you that it's very fun and rewarding. The difficult part is not writing the emulator, it's understanding how the emulated machine works. As always, code is cheap, but documentation is priceless.

Each part of this tutorial will come with all needed code, and will be associated to a single commit in the gammaboy GitHub repository. The emulator itself is going to be written in Rust. I chose this language because I wanted to learn it, and because it seemed fit for our purposes: it's low-level, but it still has nice abstractions inspired by functional languages.
